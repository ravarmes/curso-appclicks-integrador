Estudo de Caso 6 — CRUD Completo

Objetivo da Semana

Configurar o projeto do Firebase no FlutterFlow, criando as coleções de fato e conectando a autenticação. É nesta semana que o projeto começa a salvar, alterar e excluir dados num banco de dados real, aplicando os conhecimentos planejados semana passada.

Passo a Passo da Atividade

1
Configuração Inicial Firebase Console / FlutterFlow

Configurar o projeto no console do Firebase e importá-lo nas configurações do FlutterFlow. Criem as coleções exatamente como planejado e ativem a autenticação por E-mail/Senha.

2
Implementem as Actions do CRUD FlutterFlow Actions

Em cada botão responsável pelas operações do sistema nas telas construídas, usem a "Action Chain" (Corrente de Ações): Validate Form -> Firestore Action (Create/Update/Delete) -> Navigate To / Back.

3
Preencham o Banco com Seeds (Dados Iniciais) Firebase Console

Cadastrem dados genéricos iniciais manualmente no Firestore para possibilitar que os testes funcionem adequadamente e tenham o que buscar nas telas futuras.

4
Entreguem o Mapeamento Técnico e Prints Word / Docs

Tire prints das Collections criadas no Firebase. Faça um pequeno descritivo mostrando como as ações (Botões de Confirmar ou Excluir) foram configuradas.

📄
Consolidação no Template Oficial Google Docs

Para concluir as atividades desta semana, reúna todas as informações e artefatos gerados nos passos anteriores e preencha o template oficial de entrega. Faça uma cópia do documento para a sua conta Google e compartilhe-o com o professor.

📱 Estudo de Caso: FinanCampus

Veja as implementações do CRUD e dados iniciais que o grupo do FinanCampus fez nesta semana.

Configuração do Projeto no Firebase:

  • Projeto financampus-app criado no Console do Firebase.
  • Autenticação por E-mail/Senha ativada.
  • Integração feita no FlutterFlow via "Connect Firebase Project".

Implementação do CRUD:

  • Botão "Registrar" de Nova Transação executa Action Chain: 1) Validate Form -> 2) Create Document (transacoes) -> 3) Navigate Back.

Seeds Iniciais (Categorias):
